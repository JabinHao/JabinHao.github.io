interface_stability安装流程

1. 环境安装

1.1 安装Anaconda

  1. 下载安装包

    • 下载地址：官网地址 或 清华大学镜像（推荐）
    • 选择合适的版本：
    • anaconda安装包较大，安装后要占用数个G的空间，硬盘空间不足可以考虑安装Miniconda，下载地址：清华源

  2. 安装：

  3. 安装完成：开始菜单出现以下程序

  4. 换源

    • anaconda软件源在国外，国内访问速度较慢，可以将软件源改为清华源
    • 首先需要一个代码编辑器，可以使用sublime text3、vscode、写字板（不推荐）
    • 打开用户目录下的 .condarc文件（conda配置文件），用户目录为 C:\Users\你的用户名
    • 在文件中加入以下内容：
      channels:
- defaults
show_channel_urls: true
channel_alias: https://mirrors.tuna.tsinghua.edu.cn/anaconda
default_channels:
- https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
- https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
- https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
- https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/pro
- https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/msys2
custom_channels:
conda-forge: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
msys2: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
bioconda: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
menpo: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
pytorch: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
simpleitk: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud

1.2 安装pymatgen库

  1. 检验环境
    • 找到开始菜单的 Anaconda Powershell Prompt (Anaconda3)程序，右键以管理员身份运行
    • 在窗口中输入以下命令，查看python版本
      python -V
      显示（版本不一样问题不大）：
  2. 安装pymatgen库
    • pymatgen 官方安装文档（参考）
    • 还是上一步的命令行窗口，输入以下命令，回车执行
      conda install --channel conda-forge pymatgen
    • 上一步如果一直卡着不动，ctrl+c退出，执行以下命令：
      conda install --yes numpy scipy matplotlib
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pymatgen
    • 如果报错，提示缺少C++依赖，则需安装相关组件（见下一节）
  3. 安装完成后，校验一下
    python
import pymatgen
exit()

1.3 安装C++

  1. 双击安装程序 vs_Community.exe，等待其下载完成

  2. 选择第一项，点击右下角安装

    需要联网下载，时间较长

  3. 安装完成后，重启计算机，回到步骤 1.2，继续安装pymatgen

2. 正式安装

2.1 下载源码

  1. 到github上下载源码，可以注册一个GitHub账号，等会儿可以用于登录 materials project

  2. 下载

    • 方式一：安装git，在git bash中通过以下命令下载源码

      git clone https://github.com/mogroupumd/interface_stability.git

      会得到一个名为interface_stability的文件夹

    • 方式二：点击 Download ZIP，下载后解压，得到interface_stability-master文件夹

2.2 安装

  1. 打开Anaconda Powershell Prompt (Anaconda3)程序，进入刚刚解压得到的文件目录（这里假设放在D盘下）
    cd d:\interface_stability
  2. 执行ls命令查看所有文件，如果有setup.py则可以安装
    ls
python setup.py install
  3. 测试是否安装成功
    python
from interface_stability import singlephase
  4. 再打开一个命令行窗口，执行命令：
    phase_stability -h
    出现以下提示信息则可以用

2.3 配置

  1. 打开用户目录下的.pmgrc.yaml文件（没有就新建一个，用文本编辑器先新建文件，保存的时候重命名）

  2. 登录 Materials Project网站，复制你的 API keys

  3. 在 .pmgrc.yaml 文件中加入以下内容

    PMG_MAPI_KEY: 你的API keys

  4. 完成

3. 使用

3.1 官方教程

  1. github上的说明：interface_stability
  2. 安装目录下的 README.md，markdown文件，可以用文本编辑器打开看原始格式，要看渲染后的格式可以用 MarkdownPad 或 typora

3.2 简单使用

  1. 打开Anaconda Powershell Prompt (Anaconda3)程序
  2. 输入以下命令
    phase_stability evolution Li3PS4 Li
    • Li3PS4：要算的材料
    • Li：open element，即结果横坐标是该元素化学势
    • 电化学窗口：图中的平台对应的化学势，换算后即为电化学窗口

3.3 补充

  1. 化学式与电极电位
    $$
    \mu_{Li}(\phi) = \mu^0_{Li} - e\phi
    $$
    • $\mu_{Li}(\phi)$ ：Li的电化学势
    • $\mu^0_{Li}$：Li元素化学势
    • $\phi$：电极电压
  2. 电化学窗口
    • 计算对Li电位，则将$\mu^0_{Li}$当作零点
    • Li 的e 为 1+
    • 从而有：$\phi$ = -$\mu_{Li}(\phi)$
